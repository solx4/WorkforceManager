# Views

هنا هتتبنى شاشات الواجهة الفعلية، كل واحدة UserControl مستقلة:

- `WorkersView.xaml` — البحث عن عامل وعرض/إدارة مهاراته
- `ProductsView.xaml` — إدارة المنتجات ومراحلها وأسعارها
- `DailyEntryView.xaml` — إدخال عدد القطع اليومي لكل عامل
- `ReportsView.xaml` — عرض التقارير اليومية والتقييم

هتُبنى في الخطوة التالية بعد التأكد من سلامة طبقة البيانات (Core + Data + Business).
